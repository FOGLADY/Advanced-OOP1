About Me

I am a Computing student with a strong interest in software engineering, cybersecurity and object-oriented system design. I am particularly interested in developing secure, maintainable and scalable software systems.

My technical experience includes Python, Java, C++, Linux, virtualisation, software testing and cybersecurity concepts. My professional goal is to continue developing my knowledge of secure software engineering, advanced testing and AI-aware software architecture.

Module Overview and Learning Outcomes

The module explored advanced object-oriented programming, SOLID principles, software design patterns, secure coding, concurrency, testing and scalable software architecture. It also considered how object-oriented systems can be designed to support AI models and data-oriented applications.

Learning Outcomes

  • Apply advanced object-oriented principles to solve complex software problems.
  • Use design patterns to create reusable, maintainable and flexible code.
  • Design secure and robust software architectures suitable for larger-scale systems.
  • Develop software solutions that are adaptable for AI models and efficient for data-oriented tasks.

The practical evidence for these outcomes is presented on the Units page and through the Capstone Project.

Overall Reflection

The module helped me move beyond writing code that simply produces the correct output. I developed a stronger understanding of how software structure affects maintainability, extensibility, security and testability.

The practical activities demonstrated the value of separating responsibilities and depending on abstractions instead of concrete implementations. Refactoring code according to SOLID principles showed how tightly coupled software can be transformed into smaller, reusable and more testable components.

I also learned that additional abstraction is not always beneficial. Design patterns should be selected because they address a genuine design problem, rather than being added only to make a small system appear more advanced.

The Capstone Project allowed me to combine these ideas in a larger practical context. It strengthened my independent problem-solving skills and my ability to analyse technical challenges, test solutions and improve an implementation iteratively.
