About Me

I am a Computing student with a strong interest in software engineering, cybersecurity and object-oriented system design.

My technical interests include Python, Java, C++, Linux, virtualisation, software testing and secure software development. My professional goal is to develop secure, maintainable and scalable software systems.

Module Overview and Learning Outcomes

This module focused on advanced object-oriented programming, secure coding, SOLID principles, reusable design patterns, software testing, concurrency and scalable software architecture.

  • Apply advanced object-oriented principles to solve complex software problems.
  • Use design patterns to create reusable, maintainable and flexible code.
  • Design secure and robust software architectures suitable for larger systems.
  • Develop software solutions that can support AI models and data-oriented workloads.

Reflection

The module helped me move beyond writing code that simply works. I learned to consider how class responsibilities, dependencies, abstractions and design patterns affect maintainability, testability and scalability.

The practical exercises also demonstrated the importance of selecting patterns only when they solve a genuine design problem. This improved my ability to justify technical decisions and recognise the difference between useful abstraction and unnecessary over-engineering.

Professional Skills and Action Plan

The module strengthened my object-oriented design, refactoring, problem-solving and software testing skills. It also improved my understanding of maintainable architecture and secure coding.

My next actions are to develop more advanced automated tests, improve my knowledge of secure software engineering and apply AI-aware architectural patterns in future projects.
